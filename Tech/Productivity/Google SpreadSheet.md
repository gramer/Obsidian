# 부가기능

## AppSheet

Google SpreadSheet 와 연결하여 mobile app 을 빠르게 만들 수 있는 기능

![[app-sheet-example.png]]

### Tip

- 첫번째 Row 는 헤더인 경우에 쉽게 작성이 가능하다.
- 컬럼의 데이터 타입이 자동으로 설정된다.
- Virtual Column 을 추가할 수 있다.
- 규칙에 따라서 UX 를 조정할 수 있다.
